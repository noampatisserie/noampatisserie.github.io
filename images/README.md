## 📥 איך להוריד תמונות מאינסטגרם?

### אופציה 1: דרך הטלפון
1. פתח את האינסטגרם
2. עבור לפוסט
3. לחץ על שלוש הנקודות → Save (שמור)
4. התמונה נשמרת בגלריה
5. העבר למחשב דרך Google Photos / AirDrop / USB

### אופציה 2: דרך כלי אונליין
1. גש ל-[Instaloader](https://instaloader.github.io/) או [SnapInsta](https://snapinsta.app/)
2. הדבק את קישור הפוסט
3. הורד את התמונה

### אופציה 3: צילום מסך (לא מומלץ)
- רק אם אין אפשרות אחרת
- הקפד על איכות גבוהה

## 🎯 טיפים לתמונות מושלמות

1. **תאורה טובה** - צלם באור טבעי או תאורה רכה
2. **רקע נקי** - עדיף לבן, קרם או עץ
3. **זווית** - קצת מלמעלה (45°) נראה הכי טוב
4. **עריכה קלה** - הגברת בהירות וחדות
5. **עקביות** - כל התמונות באותו סגנון
